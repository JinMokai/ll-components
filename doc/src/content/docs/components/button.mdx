## 插槽

:::note[笔记📖]

通过使用```slot='prefix'```，你可以将元素插入到按钮的前面。
默认的书写文本是默认插槽。
通过使用```slot='suffix'```，你可以将元素插入到按钮的后面。

```html
<!-- index.html -->
  <ll-button>
    <span slot='prefix'>😀</span>
    default
  </ll-button>
  <ll-button>
    😀
  </ll-button>
  <ll-button>
    <span slot='suffix'>😀</span>
    default
  </ll-button>
```
:::

  component={`

  `}

## 怎么修改按钮的样式

:::note[笔记📖]
我们可以使用[::part()](https://developer.mozilla.org/en-US/docs/Web/CSS/::part)语法，通过这个伪类可以穿透shadow dom盒子。
::part()伪类可以匹配的按钮有```button```,```prefix```,```label```,```suffix```,分别是按钮自身，按钮前缀，按钮文字，按钮后缀。
:::

```css
/* index.css */
.custom-class::part(button) {
  background-color: #f00;
  color: #fff;
}
```

<Preview client:load 
  component={`
  <ll-button class='custom-class'>default</ll-button>
  `}
