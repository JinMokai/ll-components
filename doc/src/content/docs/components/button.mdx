## 怎么修改按钮的样式

我们可以使用::part()语法，通过这个伪类可以穿透shadow dom盒子。
::part()伪类可以匹配的按钮有```button```,```prefix```,```label```,```suffix```,分别是按钮自生，按钮前缀，按钮文字，按钮后缀。

```css
.custom-class::part(button) {
  background-color: #f00;
  color: #fff;
}
```

<Preview client:load 
  component={`
  <ll-button class='custom-class'>default</ll-button>
  `}
/>
